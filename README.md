## Building the program
1. Clone this repo.

```
git clone https://github.com/elioth-frutos/reversi_entrega
```

2. `cd` into the root directory and run `make`.

```
 cd reversi_entrega
 make
```

## Running the program
After building the source, run `./reversi` from the project root.
